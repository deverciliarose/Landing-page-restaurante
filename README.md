# Landing Page - Sabor & Brasa
Uma landing page moderna desenvolvida para apresentar um restaurante fictício especializado em comida na brasa.
O projeto foi criado com foco em desenvolvimento frontend.

Você pode visualizar o projeto online clicando no link abaixo:
[** Acesse o Sabor & Brasa aqui **](https://landing-page-restaurante-alpha.vercel.app)
### ✨ Funcionalidades
*Layout moderno e responsivo
*Navbar com navegação entre seções
*Seção de apresentação do restaurante
*Cards de produtos/pratos 
*Animações ao rolar a página
*Design adaptado para dispositivos móveis
### 🚀 Objetivos do projeto
Esse projeto foi desenvolvido com objetivo de:
*Praticar desenvolvimento frontend
*Aprimorar habilidades em HTML, CSS e JavaScript
*Criar um projeto para portfólio
### 📱 Responsividade
A landing page foi desenvolvida seguindo o conceito mobile-first, garantindo boa experiência em:
*Celulares
*Tablets
*Desktop
